# 아나콘다 주요 커맨드

## 가상 환경 목록 확인

```bash
$ conda env list
```

## 가상 환경 생성

```bash
$ conda create -n <envname>
$ conda create -n <envname> python=<version>
```

## 가상 환경 활성화 & 비활성화

```bash
$ conda activate <envname>
$ conda deactivate
```

## 가상 환경 삭제

```bash
$ conda env remove -n <envname>
```

## 가상 환경에 설치되어 있는 패키지 목록 확인

```bash
$ conda list
```
