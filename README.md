- TIL은 "Today I Learned"의 약자로, 매일 새로운 지식이나 경험을 학습하고 기록하는 것을 의미합니다. 
- 이 저장소는 제가 하루하루 학습한 내용을 정리하여 기록함으로써 성장의 흔적을 남기기 위한 공간입니다.
